# Short Notes Generation Checklist

One row per CFA Level I reading. Tick a column once that flavour of short notes is generated, reviewed, and approved into `study_notes`.

**Kinds**
- **F** = Formulas (definitions, symbols, units, pitfalls)
- **T** = Terms (vocabulary, definitions, contrasts)
- **C** = Concepts (frameworks, relationships, intuition, traps)

**Workflow**
1. Pick an unchecked reading.
2. Open the corresponding CFA volume PDF in `/CFA material/`.
3. Run the notes generator at `/notes/generate`, pick the chapter, choose kind, add extra-topic prompt if needed.
4. Review the draft batch. Approve or redo prompt.
5. Tick the box here. One reading at a time — do not batch-generate the whole syllabus.
